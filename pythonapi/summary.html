get_nuclide_by_zaid(zaid)¶

Return a Nuclide object given the ‘zaid’ identifier for the nuclide.

Parameters:zaid (int) – 1000*Z + A, where Z is the atomic number of the nuclide and A is the mass number. For example, the zaid for U-235 is 92235.
Returns:nuclide – Nuclide matching the specified zaid, or None if no matching object is found.
Return type:openmc.nuclide.Nuclide or None
