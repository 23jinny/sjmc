Summary¶

class openmc.summary.Summary(filename)¶

Information summarizing the geometry, materials, and tallies used in a simulation.

openmc_geometry¶

openmc.Geometry

An OpenMC geometry object reconstructed from the summary file

opencg_geometry¶

opencg.Geometry

An OpenCG geometry object equivalent to the OpenMC geometry encapsulated by the summary file. Use of this attribute requires installation of the OpenCG Python module.
