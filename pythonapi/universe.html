Universe¶

class openmc.universe.Cell(cell_id=None, name='')¶

A region of space defined as the intersection of half-space created by quadric surfaces.

Parameters:
  • cell_id (int, optional) – Unique identifier for the cell. If not specified, an identifier will automatically be assigned.
  • name (str, optional) – Name of the cell. If not specified, the name is the empty string.
id¶

int

Unique identifier for the cell

name¶

str

Name of the cell

fill¶

Material or Universe or Lattice or ‘void’

Indicates what the region of space is filled with

region¶

openmc.region.Region

Region of space that is assigned to the cell.

rotation¶

ndarray

If the cell is filled with a universe, this array specifies the angles in degrees about the x, y, and z axes that the filled universe should be rotated.

translation¶

ndarray

If the cell is filled with a universe, this array specifies a vector that is used to translate (shift) the universe.

offsets¶

ndarray

Array of offsets used for distributed cell searches

add_surface(surface, halfspace)¶

Add a half-space to the list of half-spaces whose intersection defines the cell.

Parameters:
  • surface (openmc.surface.Surface) – Quadric surface dividing space
  • halfspace ({-1, 1}) – Indicate whether the negative or positive half-space is to be used
