Parameters:
  • particles (int) – Number of particles to simulate per generation
  • threads (int) – Number of OpenMP threads
  • geometry_debug (bool) – Turn on geometry debugging during simulation
  • restart_file (str) – Path to restart file to use
  • tracks (bool) – Write tracks for all particles
  • mpi_procs (int) – Number of MPI processes
  • output (bool) – Capture OpenMC output from standard out
  • openmc_exec (str) – Path to OpenMC executable
