Quick Install Guide¶

This quick install guide outlines the basic steps needed to install OpenMC on your computer. For more detailed instructions on configuring and installing OpenMC, see Installation and Configuration in the User’s Manual.

Installing on Ubuntu through PPA¶

For users with Ubuntu 11.10 or later, a binary package for OpenMC is available through a Personal Package Archive (PPA) and can be installed through the APT package manager. Simply enter the following commands into the terminal:

sudo apt-add-repository ppa:paulromano/staging
sudo apt-get update
sudo apt-get install openmc

Currently, the binary package does not allow for parallel simulations or use of HDF5. Users who need such capabilities should build OpenMC from source as is described in Installation and Configuration.

Installing from Source on Linux or Mac OS X¶

All OpenMC source code is hosted on GitHub. If you have git, the gfortran compiler, and CMake installed, you can download and install OpenMC be entering the following commands in a terminal:

git clone https://github.com/mit-crpg/openmc.git
cd openmc
git checkout -b master origin/master
mkdir build && cd build
cmake ..
make
sudo make install

This will build an executable named openmc and install it (by default in /usr/local/bin). If you do not have administrator privileges, the cmake command should specify an installation directory where you have write access, e.g.

cmake -DCMAKE_INSTALL_PREFIX=$HOME/.local ..
